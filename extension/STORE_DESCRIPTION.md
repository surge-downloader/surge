# Surge Browser Extension

## Short Description
Integrate with Surge Download Manager. Intercepts downloads to accelerate them using Surge's high-performance engine.

## Long Description
Surge is a blazing fast, open-source download manager. This extension integrates your browser with Surge, automatically intercepting download links and sending them to the Surge engine for accelerated downloading.

**Features:**
- **Seamless Integration**: Automatically detects and intercepts downloads.
- **High Performance**: Offloads downloads to Surge which uses multi-connection downloading, work stealing, and smart chunking to maximize speed.
- **Privacy Focused**: Open source and transparent.
- **TUI & Headless Support**: Works with Surge in both Interactive TUI and Headless Server modes.

**Requirements:**
- You must have the [Surge Download Manager](https://github.com/surge-downloader/surge) installed and running on your system.
